## ✨ What Makes This Different?

Most icon themes are minimal to the point of being useless—slapping the same yellow square on every JavaScript variant. **VS Seti Plus** sits exactly where you need it — recognizable at a glance, detailed where it matters, and covering more file types than any theme you've used before.
VS Seti Plus is engineered by a Verified Publisher with 10+ years of experience in massive, professional codebases.

### 🎯 Over 200 Hand-Crafted Icons

Stop reading filenames to understand your architecture. This system provides distinct visual identities that most packs ignore:

- Zero-Ambiguity JS: Total clarity between JS, MJS, CJS, and OJS.

- Web Standards: Precise differentiation between XML, HTML and HTM.

- Python Specialization: Unique icons for Scripts vs. Modules vs. Handlers using our signature 8-color functional spectrum.

- React support: Subtle difference between JSX and TSX icons is clear even at small scale.

> The **4px Recognition Rule**: Every icon is hand-tuned for high-contrast visibility. Whether your sidebar is wide or collapsed to a 4x4px scale, the silhouette and color remain distinct and recognizable.

### 🎨 Bright enough to be visible, soft enough not to hurt

**VS Seti Plus** uses a deliberate palette of bright Red, Orange, Yellow, Green, Blue, Purple, Pink, and White. These aren't just colors — they are high-contrast tones designed to pop against dark backgrounds and reduce "search time" during long coding sessions.

## 🚀 Setup

### Installation

1. Open VSCode
2. Go to Extensions (`Ctrl+Shift+X` / `Cmd+Shift+X`)
3. Search for **"VS Seti Plus"**
4. Click **Install**

### Activate the Icon Theme

1. Open Command Palette (`Ctrl+Shift+P` / `Cmd+Shift+P`)
2. Type **"File Icon Theme"**
3. Select **"Seti Plus (Dark)"**

### Activate the Color Theme

1. Open Command Palette (`Ctrl+Shift+P` / `Cmd+Shift+P`)
2. Type **"Color Theme"**
3. Select whichever color theme you'd prefer

## 🐛 Known Issues

- Widget translucency without blur: Without the optional CSS loader installed, editor widgets (suggest, hover) will be translucent but will not have the `backdrop-filter: blur()` effect. This is a VSCode limitation, not a theme bug. See Enhanced Visuals above.

- Terminal cursor color: Some terminal configurations may override the cursor color. Set `"terminal.integrated.cursorStyle": "line"` if the cursor isn't visible.
